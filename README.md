# API Inteligente com Spring Boot e IA

## 🚀 O que o projeto faz
Este projeto implementa uma API de orçamento inteligente usando **Spring Boot** e **Spring AI**.  
Ele permite criar e consultar transações financeiras a partir de comandos de voz, integrando 
recursos de **transcrição de áudio**, **interpretação de intenção com IA** e **resposta em áudio**.

## 🔄 Fluxo principal
1. Receber um arquivo de áudio enviado pelo cliente.
2. Transcrever o áudio em texto.
3. Usar IA para entender a intenção do comando.
4. Executar uma função real da aplicação (criar ou consultar transações).
5. Persistir os dados no banco.
6. Retornar a resposta final em texto ou áudio.
